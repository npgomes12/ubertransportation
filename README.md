<h1>Uber Ride Data Analysis and Cleaning Project</h1>

<h2>Description</h2>

This project consists of extracting, cleaning, transforming, and analyzing Uber ride data from Boston, Massachusetts, using Python, Pandas, and SQL. The dataset was sourced from Kaggle and prepared for use by transportation planners, civil engineers, and urban planning committees. The project explores ride prices, trip distances, demand patterns, weather conditions, and ride characteristics while identifying trends and potential biases within the data.
